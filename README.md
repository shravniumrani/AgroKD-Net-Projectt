# 🌱 AgroKD-Net: Smart Crop-Weed Detection

## 📌 Overview
AgroKD-Net is a lightweight deep learning model designed for real-time crop and weed detection in agricultural fields. The model combines YOLO-based detection with knowledge distillation to improve efficiency and accuracy.

---

## 🚀 Features
- Real-time object detection
- Lightweight and efficient model
- Suitable for edge devices and drones
- Handles small object detection

---

## 📊 Results

### 🔹 Model Comparison
| Model | mAP | Precision | Recall | Inference Time (ms) |
|------|-----|----------|--------|---------------------|
| YOLOv5 | 0.82 | 0.85 | 0.83 | 150 |
| YOLOv7 | 0.85 | 0.87 | 0.86 | 180 |
| MobileNet-SSD | 0.78 | 0.80 | 0.79 | 210 |
| EfficientDet | 0.86 | 0.88 | 0.87 | 220 |
| Faster R-CNN | 0.88 | 0.89 | 0.88 | 600 |
| AgroKD-Net | 0.87 | 0.90 | 0.89 | 160 |
